# Bank Management System (C++)

A console-based banking system developed using C++ that allows managing client accounts and performing banking transactions using file storage.

## Features

### Client Management
- Add new clients
- View all clients
- Search for clients by account number
- Update client information
- Delete clients

### Banking Transactions
- Deposit money
- Withdraw money
- Display total balances of all clients

### Data Management
- Save client records to files
- Load client data from files
- Check duplicate account numbers before adding clients

## Programming Concepts Used
- Object-Oriented Programming Concepts
- Structs
- File Handling
- Vectors
- Enums
- Functions
- String Processing
- Modular Programming

## Technologies
- C++
- Console Application
- Visual Studio
